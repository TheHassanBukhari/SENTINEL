SENTINEL

Spread and Exposure Network for Tracking, Isolation and Neutralizing Epidemic Linked Nodes

A console-based epidemic contact tracing system built entirely in C++, with every data structure implemented from scratch using linked lists.

Data Structures and Algorithms · 3rd Semester · Spring 2025

About this project

What SENTINEL is

SENTINEL models a disease outbreak as a live graph of people and contacts. It tracks patients, calculates transmission risk, simulates how an outbreak spreads wave by wave, traces the most probable infection chain from patient zero to any individual, and identifies the minimum set of people whose quarantine would collapse an entire cluster. Every structure has one specific job that nothing else in the system can do.

The problem it solves
When an epidemic hits, health authorities need to know who infected whom, who is highest risk right now, and where to intervene surgically rather than quarantining everyone blindly. SENTINEL answers all three using graph algorithms and priority-based data structures.
What makes it different
Every patient carries their own private investigation stack. Every contact event has a weighted transmission risk. Dijkstra finds the most probable infection chain. Prim's finds the minimum containment cut. None of this is decorative.
Implementation constraint
No STL containers anywhere. Every stack, queue, heap, tree, and hash table is built node by node using raw linked list logic in C++. The constraint was the point: understanding what these structures actually are, not just calling them.
Data
50 patients across four area zones, 65 contact events with risk weights based on duration, location type, and proximity. Loaded at startup from two CSV files into all seven structures simultaneously.

Structures and algorithms used
Stack
Per-patient investigation log. LIFO event history.
Queue
Master case pipeline and BFS wave simulation.
Heap
Risk priority engine. Highest risk always on top.
AVL Tree
Patient records indexed by day. Range search.
HashMap
Dual lookup: patient profiles and area hotspots.
Graph
Weighted contact network. Foundation of all analysis.
Dijkstra
Shortest transmission path from patient zero.
Prim's
Minimum containment cut across a cluster.
